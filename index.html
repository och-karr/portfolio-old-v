O mnie

Cześć! Jestem Karolina i mieszkam w Krakowie. Tworzeniem stron zaczęłam interesować się pod koniec 2017 roku i od tamtej pory rozwijam swoje umiejętności w tej dziedzinie. Na początku samodzielnie rozwijałam swoje umiejętności, ale później zdecydowałam się na udział w półrocznym bootcampie Kodilli. Obecnie, już po kursie, samodzielnie rozwijam swoje umiejętności poprzez czytanie książek związanych z tematem, ze szczególnym naciskiem na JavaScript - aby lepiej zgłębić jego tajemnice oraz robieniem małych projektów. W lutym podjęłam staż na stanowisku Junior Front-end Developera. Efekty mojej dotychczasowej pracy można zobaczyć na tej stronie. Zapraszam również do odwiedzenia mojego Githuba.

Technologie

W swoich projektach wykorzystuję najczęściej HTML, CSS oraz JavaScript, dlatego czuję się najlepiej w tych technologiach. JavaScript to bardzo obszerny i złożony temat, dlatego sporą część mojego czasu poświęcam na poszerzanie wiedzy w tematyce tego języka. W moich projektach wykorzystywałam również jQuery (np. Kanban board), Ajax API (Country Search Engine) oraz Sass (RWD website albo przy wykonywaniu tej strony). Znana jest mi biblioteka Bootstrap oraz program Avocode (analiza projektu graficznego w formacie psd, export grafik). Podczas kursu miałam również do czynienia z Reactem (przykładowy projekt dostępny tutaj ). Wiedzę z Reacta pogłębiam w tym momencie robiąc kurs na Udemy – efekt mojej pracy można zobaczyć tutaj. Nauczyłam się również podstaw PHP, które wykorzystałam do wykonania formularza.

Na stażu uczyłam się nauczyłam się również korzystać z platformy ecommerce - Shopware oraz task managera - Gulp. Podczas stażu wykonałam dwa projekty: theme sklepu internetowego (nadal w trakcie prac) z wykorzystaniem platformy ecommerce - Shopware oraz projekt responsywnej landing page z wykorzystaniem HTML, Sass, JavaScript oraz task managera Gulp.
